# Housing Value Prediction Using Linear Regression

## Project Overview

This project applies multiple linear regression techniques to predict residential property values using the West Roxbury Housing and Boston Housing datasets.

The analysis includes data preprocessing, feature engineering, model training, validation, and performance evaluation using industry-standard regression metrics.

## Tools and Technologies

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Jupyter Notebook

## Methods Used

* Data Cleaning
* One-Hot Encoding
* Train/Test Data Splitting
* Multiple Linear Regression
* Residual Analysis
* Model Validation

## Evaluation Metrics

* Root Mean Squared Error (RMSE)
* Mean Absolute Error (MAE)
* Mean Absolute Percentage Error (MAPE)
* Mean Error (ME)

## Key Findings

The regression models successfully predicted housing values with relatively low prediction error across both datasets. Model performance was evaluated using separate training and validation datasets to assess generalization and reduce overfitting risk.
